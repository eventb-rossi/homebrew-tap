# Homebrew Tap for Event-B Rossi

Install the tap:

```sh
brew tap eventb-rossi/tap
```

Install the Rodin Platform (macOS app):

```sh
brew install --cask rodin
```

[Rodin](https://www.event-b.org/) is the Eclipse-based IDE for formal modelling and
verification with Event-B.

> Rodin 3.9 ships only an Intel (x86_64) build. On Apple Silicon it runs under Rosetta 2.
